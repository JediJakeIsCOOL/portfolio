StockMuncher

Project Summary

StockMuncher is a stock information website that was created using backend coding and databases. Some of the technologies used includes Node.js, EJS, postgreSQL, and various stock api calls. This website includes a functioning signup and login page. It stores passwords as encrypted hashes and creates sessions for users who sign in. The stock searchbar auto-fills and lets you view current prices of the stock. It also creates a graph of the current stock selected.
