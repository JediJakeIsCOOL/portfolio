Project Summary

The Activity Finder app was created using HTML, CSS, Boostrap, JQuery, API calls, and JavaScript. The app is able to take in categorie constraints and a location in order to find you activities in your near vicinity. The weather is updated for the location you use. Twenty events and activities are updated for the area and categories you choose, each linking to more information about the event or activity. Upon entering your location, the search bar autofills and a google map of the events and activities with locations marked appears on the webpage.

Interesting Code Used

The Google Places API was used to retrieve information on specific activities in a city. Used this autocomplete function in order to automatically finish the names of cities that you are searching for when looking for activities in your specific location.

The Google Maps API was used to map the data that was received. Using the DOM along with the API callbacks from the Google maps API, the app is able to locate the different longitude and latitude of events. Centering itself on the city of your choice and showing pin markers of the events in around your locotion depending on the category criteria.
